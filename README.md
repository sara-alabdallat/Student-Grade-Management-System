# Student Grade Management System

A Python-based project designed to manage student grades efficiently using fundamental data structures and algorithms.

## 📌 Overview
This system allows users to input student names and grades, automatically sort them using Merge Sort, and search for specific students using Binary Search.

## 🧠 Algorithms Used
- Merge Sort (sorting grades in descending order)
- Binary Search (searching student names efficiently)

## ⚙️ Key Features
- Dynamic data entry until user types 'stop'
- Sorting students by highest grade
- Case-insensitive search using `.lower()`
- Efficient searching after sorting

## 🎯 Purpose
To demonstrate how algorithms can be applied in real-world data management systems.
